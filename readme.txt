Project Overview

The Email Categorization Predictor is a machine learning-based web application that classifies emails into different categories, such as Personal, Work, Spam, and Promotions. The frontend is built using HTML, CSS, and JavaScript, while the backend runs on Flask with a trained ML model for classification.

 Features

Classify emails into different categories.
Fast and lightweight prediction system.
Web-based UI for user-friendly interaction.
REST API for email classification.


 Installation & Setup

Prerequisites

Python (>=3.8)
Flask


Required dependencies (listed in requirements.txt)

Clone the Repository


Install Dependencies

pip install -r requirements.txt


Run 

python datasets_processing.py (train model)
python app.py

Open the Web App

Enter an email and click Classify to get a prediction.
